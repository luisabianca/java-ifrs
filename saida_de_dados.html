Saída de Dados

A última instrução do programa de cálculo da média é:

System.out.println(RES);

Esta instrução permite que se mostre algo na tela do terminal de comandos.
É possível exibir, em uma única instrução de saída várias linhas de texto. Para isso utilizamos o caractere especial de nova linha, representado pela sequência “\n”.
Quando esta combinação é encontrada, o cursor é posicionado em uma nova linha e a exibição prossegue a partir do próximo caractere.
A figura abaixo mostra um programa que mostra a mensagem “Bem-vindo ao mundo Java!” em várias linhas. A saída deste programa é mostrada logo após.

public class BemVindo {
  public static void main (String[] args) {
    System.out.println("Bem-vindo\nao\nmundo\nJava!");
  }
}

Resultado:
Bem-vindo
ao
mundo
Java!
