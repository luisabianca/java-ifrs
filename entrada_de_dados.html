Entrada de Dados

Em Java existem diversas maneiras de realizar entrada de dados.
Uma delas é através do método System.console().readLine().

Este método para a execução do programa enquanto o usuário não digitar algo pelo teclado e pressionar a tecla ENTER.
Quando isso acontece, o método devolve o que foi digitado ao programa como um valor String. Esse valor não é armazenado em uma variável automaticamente.
Por isso precisamos utilizar o operador de atribuição para colocar o valor digitado em alguma variável, como fizemos no programa para cálculo da média de notas.

O que é Double.parseDouble? Este é outro método que existe em Java. Ele serve para transformarmos um valor String em double.
Nós o utilizamos em nosso programa do cálculo da média para transformar o valor devolvido por System.console().readLine(), que é String, em um valor double, que pode ser armazenado nas variáveis N1, N2, N3 e N4 (todas do tipo double). Isso foi necessário porque não podemos armazenar um valor String em uma variável double.

Outros exemplos de conversão:

Boolean.parseBoolean
Integer.parseInt
Byte.parseByte
Float.parseFloat
Long.parseLong
Short.parseShort

Note que não existe um método de conversão para o tipo char. Como fazer para ler dados desse tipo? Pode-se fazer a leitura de um valor String e depois extrair o primeiro caractere dessa String, utilizando o método charAt, como mostrado abaixo:

System.console.readLine().charAt(0)
