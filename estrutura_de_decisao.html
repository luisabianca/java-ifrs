Estrutura de Decisão do tipo Se...Então

se teste entao
conjunto de instrucoes 1
senao
conjunto de intrucoes 2
fim se

Ordem de precedência:

  1. NÃO
  2. / * mod div
  3. + -
  4. > >= < <=
  5. = <> !=
  6. E
  7. OU

Estrutura em Java:

if (teste)
conjunto de instrucoes 1
else
conjunto de intrucoes 2

A estrutura do if deve estar sempre entre parênteses.

public class decisao {
public static void main(String[] args) {
double N1, N2, N3, N4, RES;

System.out.println("Informe a nota do primeiro bimestre: ");
N1 = Double.parseDouble(System.console().readLine());
System.out.println("Informe a nota do segundo bimestre: ");
N2 = Double.parseDouble(System.console().readLine());
System.out.println("Informe a nota do terceiro bimestre: ");
N3 = Double.parseDouble(System.console().readLine());
System.out.println("Informe a nota do quarto bimestre: ");
N4 = Double.parseDouble(System.console().readLine());

RES = (N1 + N2 + N3 + N4) / 4;

System.out.print("Média das notas é: ");
System.out.print(RES);

if(RES >= 7) {
System.out.println("Parabéns, você foi Aprovado!");
} else {
System.out.println("Infelizmente você foi Reprovado!");
}
}
}

Receber dois números e mostrar o maior

public class MaiorNumero {
public static void main(String[] args) {
double N1, N2;

System.out.println("Informe o primeiro número: ");
N1 = Double.parseDouble(System.console().readLine());

System.out.println("Informe o segundo número: ");
N2 = Double.parseDouble(System.console().readLine());

if(N1 > N2)
System.out.println(N1);
if(N2 > N1)
System.out.println(N2);
if(N1 == N2)
System.out.println("Os números são iguais");
}
}
